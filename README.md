# Portfolio

Files for the second version of my my personal portfolio, built with [Astro](https://astro.build/).

## Scripts

To start the development server:

```sh
$ npm run dev
```

```sh
$ npm run build
```

To serve a production build:

```sh
$ npm run preview
```
